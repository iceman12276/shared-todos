# Shared Todos

Test-bed project for the multi-team agentic coding system.

**Status:** scaffolded — awaiting initiative kickoff.

**Scope (to be refined by PM during planning phase):**
- User auth (register / login / logout / session)
- CRUD for todo lists and items
- Share a list with another user by username/email
- Permissioning: owner vs. shared-viewer, maybe shared-editor
- Multi-screen frontend: login, register, dashboard, list detail, share modal

**Why this feature:** small enough to ship end-to-end through the planning → engineering → validation pipeline, but complex enough to exercise every team's skills and tools (auth, authz, CRUD, multi-user attack surface, multi-page UI).
